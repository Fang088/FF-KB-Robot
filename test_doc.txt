FF-KB-Robot 测试文档
================================

这是一个用于测试 FF-KB-Robot 上传功能的示例文档。

## 一、项目简介

FF-KB-Robot 是一个功能强大的知识库机器人系统，主要功能包括：
- 文档上传与管理
- 智能文本分块与清洗
- 向量数据库存储与检索
- 自然语言查询接口

## 二、使用说明

### 1. 启动系统
```bash
python main.py
```

### 2. 创建知识库
- 输入知识库名称
- 输入知识库描述

### 3. 上传文档
使用 `upload` 命令上传文档：
```
请输入你的问题（或命令）: upload
请输入文档路径: test_document.txt
```

## 三、技术特点

1. **多格式支持**：支持 TXT、DOCX、PDF、Excel 等多种格式
2. **智能分块**：基于语义的文本分块，确保上下文完整性
3. **向量存储**：使用 ChromaDB 进行向量存储，支持快速检索
4. **可扩展性**：支持多种 LLM 模型和向量数据库

## 四、示例内容

这是一段用于测试分块功能的长文本。FF-KB-Robot 会将文本分割成大小合适的块，并保留句子的完整性。系统会自动生成向量并存储到向量数据库中。

当用户查询相关问题时，系统会将问题转换为向量，并与知识库中的向量进行相似度匹配，返回最相关的结果。
